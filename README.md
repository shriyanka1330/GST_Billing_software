# GST Billing Software

A desktop-based GST Billing Software developed using Python, Tkinter, and MySQL.

## Features
- User Login System
- Dashboard
- Customer Management
- Product Management
- GST Invoice Generation
- Billing and Payment Records
- Database Storage using MySQL

## Technologies Used
- Python
- Tkinter
- MySQL

## Project Purpose
This project is developed for learning and implementing billing management system concepts using Python.
